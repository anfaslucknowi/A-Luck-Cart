# 🛒 A-Luck Cart

A-Luck Cart is a **full-stack MERN e-commerce clothing platform** that allows users to browse products, add items to their cart, and place orders online.
The project includes a **modern user interface, secure backend APIs, and an admin panel for product management**.

This project demonstrates the implementation of a complete **E-commerce workflow using the MERN stack**, including product management, cart functionality, and order processing.

---

## 🚀 Features

* 🛍 Browse clothing products
* 🔍 View product details
* 🛒 Add products to cart
* ➕ Update or remove items from cart
* 📦 Place orders
* 🔐 Secure backend APIs
* 🧑‍💼 Admin panel for managing products
* ☁️ Image upload using Cloud storage
* 💳 Online payment integration support

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/anfaslucknowi/A-Luck-Cart.git
```

Go to the project directory:

```
cd A-Luck-Cart
```

Install dependencies:

```
npm install
```

Run the project:

```
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder and add:

```
MONGODB_URI=your_mongodb_connection
CLOUDINARY_API_KEY=your_cloudinary_key
CLOUDINARY_SECRET=your_cloudinary_secret
STRIPE_SECRET_KEY=your_stripe_key
```
